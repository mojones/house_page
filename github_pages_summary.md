# Summary

Deployed the Haddington property facilities map as a static GitHub Pages site.

## App

- Vite static single-page app.
- Interactive Leaflet map using a light CARTO basemap.
- Mobile-optimized controls for route mode, category and destination.
- Static walking/cycling route geometry generated from pedestrian/cycle routing data.
- No live route API calls required by the deployed page.

## GitHub

- Initialized this directory as a git repo.
- Created GitHub repo `mojones/house_page`.
- Pushed source to `main`.
- Built static assets with `npm run build`.
- Published `dist/` to the `gh-pages` branch.
- Enabled GitHub Pages from `gh-pages` branch root.
